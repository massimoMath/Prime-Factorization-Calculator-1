Evaluate your device's performance

This tool allows you to estimate your device's execution speed while computing prime factorization of 1x1040. More precisely, the speed which your device will take to repeat the while loops (shown below). A timer will record the amount of time the device will take to find results. Please note that this profiler will only give you an approximation, as a lot of diverse factors may unfavourably impact your score.

Here is the JavaScript function (simplified) that will be executed to prime factorize number:

      function Calculation() {
        var dividend = 1e+40;
        var divisor = 2;
        while (dividend > 1) {
          while (dividend % divisor == 0) {
            var result = dividend / divisor;
            dividend = result;
          }
        }
        divisor += 1;
      }
